


* Built a website to book local services like plumbing, electrical work, and painting all in one place instead of calling multiple providers and checking availability one by one
* Users can search service providers by their reviews, pricing and location, see which time slots are available, and book instantly with email confirmation
* Service providers can accept or reject bookings, and admins manage the entire platform by adding or removing providers
* Solves the real frustration of having to call different service people separately every time you need work done
